# Product_management_store
A full-stack Product Management System built with the MERN stack. This application features real-time updates, secure authentication, and a modern UI, allowing users to manage product inventories seamlessly.

🌟 Features:

🎃 Full Tech Stack: Built using the MERN stack (MongoDB, Express, React, Node.js).

👾 Authentication & Authorization: Secure user access implemented with JSON Web Tokens (JWT).

🚀 Real-time Updates: Instant data synchronization and messaging powered by Socket.io.

👌 User Status: Real-time tracking of online user status using Socket.io and React Context API.

👾 State Management: Clean and efficient global state handling with Zustand.

👹 Modern UI: Styled with TailwindCSS and Daisy UI for a responsive, polished look.

🎃 Error Handling: Comprehensive error boundaries and handling on both the Client and Server.

🚀 Deployment Ready: Optimized for free deployment platforms.

# 🚀 Getting Started
⭐ Prerequisites

👹 Node.js installed

👾 MongoDB account/URI

🌟 npm or yarn
### Setup .env file

```js
MONGO_URI=...
PORT=5000
JWT_SECRET=...
NODE_ENV=...
```

### Run this app locally

```shell
npm run build
```

### Start the app

```shell
npm run start
```
